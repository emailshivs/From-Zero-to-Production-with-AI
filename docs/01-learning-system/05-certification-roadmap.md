# Certification Roadmap

## Purpose

The certification roadmap defines internal readiness levels for the handbook. It does not claim external accreditation. It helps learners, instructors, and teams understand what evidence demonstrates increasing capability.

## Certification Levels

| Level | Name | Evidence standard |
| --- | --- | --- |
| Level 1 | Foundations | Orientation, core vocabulary, starter project evidence |
| Level 2 | Builder | Guided product project, labs, screenshots, and quiz evidence |
| Level 3 | Professional Practice | Development, quality, security, and AI documentation evidence |
| Level 4 | Production Ready | Operations, validation, and release-readiness evidence |
| Level 5 | Architect Portfolio | Capstone architecture, governance, and presentation evidence |

## Level 1: Foundations

Required evidence:

- Completed Getting Started checklist.
- Completed Phase 00 overview items.
- Glossary familiarity.
- One beginner quiz.
- One reflection explaining the learner’s target path.

## Level 2: Builder

Required evidence:

- Completed beginner or university student path requirements.
- One practical project.
- Required screenshots accepted.
- Lab validation evidence.
- Quiz average of 70% or higher.

## Level 3: Professional Practice

Required evidence:

- Development workflow evidence.
- Quality engineering evidence.
- Security review evidence.
- Prompt documentation evidence when AI is used.
- Capstone milestone review.

## Level 4: Production Ready

Required evidence:

- Deployment or release readiness checklist.
- Operations validation evidence.
- Incident or troubleshooting reflection.
- Security and quality evidence accepted.
- Instructor or reviewer sign-off.

## Level 5: Architect Portfolio

Required evidence:

- Completed advanced capstone.
- Architecture package.
- Skills matrix mapping.
- Review presentation.
- Traceability from business objective to validation evidence.

## Badge Criteria

If badges are introduced later, each badge must map to objective evidence. Do not issue badges for reading alone.

## Review and Renewal

Certification criteria should be reviewed with each minor release. Renewal may be required when major chapter changes alter required evidence.
