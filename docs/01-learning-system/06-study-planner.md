# Study Planner

## Purpose

The study planner helps learners and instructors select a realistic pace. Plans may be adapted for self-study, bootcamps, university courses, or team enablement.

## Planning Assumptions

- Each week includes reading, lab work, quiz review, and project evidence.
- Learners should reserve time for troubleshooting.
- Capstone work should start before all theory is complete.
- Faster plans require tighter scope and more instructor support.

## 4-Week Intensive Plan

| Week | Focus | Evidence |
| --- | --- | --- |
| 1 | Getting Started, Phase 00, path selection | Setup evidence, starter project brief |
| 2 | Phase 01 and Phase 02 essentials | Development and validation evidence |
| 3 | Phase 03 and Phase 04 essentials | Security review and prompt record evidence |
| 4 | Phase 05 overview, Phase 06 overview, capstone presentation | Final project package |

Best for bootcamps, focused teams, or experienced learners.

## Study Habits

Recommended weekly rhythm:

1. Preview objectives.
2. Read the chapter.
3. Complete the lab.
4. Capture evidence.
5. Take the quiz.
6. Update the dashboard.
7. Reflect on interview questions.
