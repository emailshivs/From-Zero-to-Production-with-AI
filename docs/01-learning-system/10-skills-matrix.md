# Skills Matrix

## Purpose

The skills matrix maps handbook concepts to theory, hands-on work, assessment, projects, real-world use, and Wealth Survivor application. It helps instructors plan coverage and helps learners explain capability growth.

## Matrix Columns

| Column | Meaning |
| --- | --- |
| Concept | Skill or knowledge area |
| Theory | What the learner must explain |
| Hands-on | What the learner must do |
| Assessment | How understanding is checked |
| Project | Where the skill is practiced |
| Real-world application | How the skill appears in professional work |
| Wealth Survivor mapping | How the skill connects to the advanced capstone |

## Matrix Usage

Use the matrix to:

- Identify missing evidence.
- Plan role-based learning paths.
- Build capstone rubrics.
- Prepare interview stories.
- Align chapters to practical outcomes.

## Maintenance Rules

Update the matrix whenever a new phase, chapter, project, or assessment changes the skills required from learners.
