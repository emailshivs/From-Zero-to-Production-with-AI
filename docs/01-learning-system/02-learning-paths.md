# Learning Paths

## Purpose

Learning paths let different audiences use the same handbook without losing a coherent sequence. Each path defines required study, optional study, estimated time, practical projects, and expected outcomes.

## Path Conventions

Chapter references use phase-level placeholders until final chapter titles are published.

- Required chapters: the learner should complete these for the path outcome.
- Optional chapters: useful extensions that can be deferred.
- Projects: portfolio or classroom evidence.
- Estimated study time: includes reading, labs, quizzes, and project work.
