# Assessment Framework

## Purpose

The assessment framework defines how learner progress is measured before detailed quizzes and rubrics are authored.

## Assessment Types

| Assessment type | Purpose | Evidence | Applies to |
| --- | --- | --- | --- |
| Knowledge check | Confirm terminology and concepts | Quiz answers | Every chapter |
| Lab validation | Confirm practical completion | Output, screenshot, checklist | Every lab |
| Failure lab review | Confirm troubleshooting ability | Diagnosis and fix evidence | Selected chapters |
| Project milestone | Confirm synthesis | Capstone artifacts | Capstones |
| Interview readiness | Confirm explanation skill | Oral or written answers | Phase reviews |
| Portfolio review | Confirm presentation quality | Final evidence package | Advanced capstones |

## Scoring Model

| Area | Suggested weight |
| --- | ---: |
| Chapter knowledge checks | 20% |
| Lab validation evidence | 30% |
| Failure lab diagnosis | 15% |
| Capstone deliverables | 25% |
| Reflection and interview readiness | 10% |

## Review Standards

Assessments must be objective, evidence-based, aligned to learning objectives, and safe for beginners. Reading alone is not sufficient for completion.
