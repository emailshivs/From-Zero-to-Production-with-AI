# From Zero to Production with AI

Welcome to the handbook platform for building, securing, and operating production-grade AI-assisted software systems.

This site is structured as a professional learning and documentation system. It currently contains the documentation architecture, editorial standards, learning system, and curriculum blueprint that will govern future handbook chapters.

## Start here

- Review the editorial standards before contributing.
- Review the learning system before planning a study path.
- Review the curriculum blueprint before drafting chapters, labs, quizzes, diagrams, or assessments.

## Deployment profile

This repository is configured as a MkDocs Material project suitable for Cloudflare Pages.

| Setting | Value |
| --- | --- |
| Build command | `pip install -r requirements.txt && mkdocs build --strict` |
| Output directory | `site` |
| Python dependencies | `requirements.txt` |
| Site configuration | `mkdocs.yml` |
